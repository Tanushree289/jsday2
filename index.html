1.Explain call(), apply() and, bind() methods. Give an example of call(), apply(), bind()

call() - It invokes the function and allows you to pass in arguments one by one.
    ex - const person = {
        firstName:"John",
        lastName: "Doe",
        fullName: function () {
          return this.firstName + " " + this.lastName;
        }
      }
      
      person.fullName();  
      
    apply() - It invokes the function and allows you to pass in arguments as an array.
    ex - const person = {
        fullName: function() {
          return this.firstName + " " + this.lastName;
        }
      }
      
      const person1 = {
        firstName: "Mary",
        lastName: "Doe"
      }
      person.fullName.apply(person1);
    
    
    
    bind() - I t returns a new function, allowing you to pass in a this array and any number of arguments.
    ex - const person = {
        firstName:"John",
        lastName: "Doe",
        fullName: function () {
          return this.firstName + " " + this.lastName;
        }
      }
      const member = {
        firstName:"Hege",
        lastName: "Nilsen",
      }
      let fullName = person.fullName.bind(member);

5.What is prototype chain

In prototype chain each object has a private property which holds a link to another object called its prototype.
That prototype object has a prototype of its own, and so on until an object is reached with null as its prototype.
